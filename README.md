## Installation

You can install the Scandinavian Embedding Benchmark (`seb`) via [pip] from [PyPI]:

```bash
pip install seb
```

[pip]: https://pip.pypa.io/en/stable/installing/
[PyPI]: https://pypi.org/project/seb/


To see more examples, see the [documentation].
